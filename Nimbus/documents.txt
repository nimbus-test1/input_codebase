
## Steps to use the files

1. **Prepare the environment:**
   - Install Terraform.
   - Set up AWS CLI and configure it with your credentials.

2. **Set up variables:**
   - Replace placeholders in the YAML file with actual values.
   - Ensure you have an existing key pair in AWS or create one and place the public key path in `public_key_path`.

3. **Initialize Terraform:**
   ```bash
   terraform init
   ```

4. **Validate configuration:**
   ```bash
   terraform validate
   ```

5. **Apply the configuration:**
   ```bash
   terraform apply
   ```
   - Review the plan and confirm to proceed.

6. **Access the instance:**
   - Use the assigned Elastic IP to access the web server via HTTP or SSH.
```
